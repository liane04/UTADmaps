## Notas para a equipa

- **Coordenadas GPS** — as coordenadas na DB são aproximadas e precisam de ser confirmadas no campus com GPS real
- **Salas** — cada edifício tem salas numeradas que precisam de ser levantadas no campus (ex: ECT-1.1, ECT-1.2...)
- **Polo I vs Polo II** — alguns edifícios têm dois polos em localizações diferentes no campus
- Fonte: mapas oficiais UTAD (`mapa-campus-simplificado-3D.pdf` e `mapa-do-campus-Congresso.pdf`)
